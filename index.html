HTML 5

HTML5 is the current iteration of hypertext markup language, which is code that describes web pages. HTML5 is usable by all Open Web developers and has included new elements, attributes and behaviors from previous versions.

CSS 3

CSS3 is the latest version of cascading style sheets, which is code that adds style to HTML documents. CSS3 allows developers to explore their artistic side by separating presentation and content.

Bootstrap

Bootstrap is a CSS framework, which allows for quick and easy formatting of HTML documents. Bootstrap is a predefined set of rules that developers can use to create asthetically pleasing layouts.

JavaScript

JavaScript is the most widely used programming language, which controls the behavior of web pages. JavaScript is also widely used for desktop and server applications. JavaScript is one of the core technologies of coding.

NodeJS

NodeJS is a javaScript run-time environment that allows us to execute javaScript outside of a browser. Node is an extremely powerful because its functions are non-blocking, allowing multiple commands to process at one time.
